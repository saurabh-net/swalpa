# 06. Sprint 5: Core Verbs & Grammar

Memorizing the verb root alongside its imperative (command) form allows for immediate action. 
**Crucial Rule:** Appending `-i` or `-ri` to the verb root ensures the request is polite. Use this default for everyone except close friends or children.

## 1. Core Action Verbs (The Engines of Communication)
These verbs handle physical movement, giving, and taking.

*   **Baa / Banni (Come):** *Banni* is the required formal application.
    *   *Usage:* "Olage banni" (Please come inside).
*   **Hogu / Hogi (Go):** Essential for directing transport.
*   **Kodu / Kodi (Give):** Used when requesting items in a shop or asking for a ticket.
*   **Thogo / Thagonli (Take):** Used when handing cash or items to a vendor.
*   **Maadu / Maadi (Do):** The universal auxiliary verb.
*   **Nodu / Nodi (See / Look):** Use it to direct someone's attention.
*   **Haaku / Haaki (Put / Drop):** Used to pour liquids or request mechanical action.
*   **Tegey / Tegeyiri (Open / Remove):**
    *   *Usage:* "Bagilu tegeyiri" (Open the door).
*   **Bidu / Bidi (Leave / Let go):**
    *   *Usage:* "Change bidi" (Keep the change).
*   **Nillu / Nillisi (Stop):** The standard respectful command for halting transport.
    *   *Usage:* "Illi nillisi" (Stop here).
*   **Iru / Iri (Be / Wait):** The existential verb.
    *   *Usage:* "Summane iri" (Just be / Keep quiet).

## 2. Communication and Data Processing
These verbs handle speech, information transfer, and instruction.

*   **Heli (Tell / Say):** Used to prompt information, like asking a vendor a final price.
*   **Keli (Ask / Listen):** Used to gain attention before a question.
*   **Kalsu / Kalsi (Send):** *"Message kalsi"*
*   **Karesu / Karesi (Call / Summon Someone):** *"Avaranna karesi"*
*   **Bary / Bareyiri (Write):** *"Hesaru bareyiri"* 
*   **Odu / Odiri (Read):** *"Idanna odiri"* 

## 3. Food and Consumption

*   **Tinnu / Tinni (Eat):**
    *   *Usage:* "Oota tinni" (Eat food).
*   **Kudi / Kudiri (Drink):**
    *   *Usage:* "Neeru kudiri" (Drink water).

## 4. Basic Sentence Forms (Present & Past)
Finite verbs in Kannada must precisely mirror the subject. 

| English Sentence | Kannada Translation | Context / Tense Focus |
| :--- | :--- | :--- |
| **I am going now.** | Naanu eega **hogthini**. | **Present (1st Person):** Root (Hogu) + Tense (-t-) + Pronoun (-ini). "I am going." |
| **They came yesterday.** | Avru ninne **bandru**. | **Past (3rd Person Plural/Respect):** Root (Baru) past tense form. "They came." |
| **I will do it tomorrow.** | Nale **maadthini**. | **Future Intent:** Present tense conjugation often acts as near future. "I will do." |
| **Look, then speak.** | **Nodi**, aamele **heli**. | **Sequential Commands:** Use the respectful imperative chained together. |

## 5. Core Conjunctions (Sentence Linking)
These words form the backbone of complex conversational Kannada.

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Mattu / Matthe** | And / Then | Used to link items (*Idli mattu Vada*) or ask "Then what?" (*Aamele matthe?*). |
| **Aadare / Adare** | But | Essential for contrast. *"Naanu bande, aadare neevu irilla."* (I came, but you weren't there). |
| **Athava** | Or | Used for options. *"Coffee athava Tea?"* (Coffee or Tea?). |
| **Yakandre / Ekeandre**| Because | Combines *Yaake* (Why) and *Andare* (If said). *"Naanu barilla, yakandre kelsa itthu."* (I didn't come because I had work). |
| **Adakke / Idarinda** | Therefore / So | *"Male banthu, adakke traffic."* (Rain came, so traffic). |
| **Hodre / Andare** | If so / Then | *"Neenu hodre, naanu barthini."* (If you go, I will come). |

## 6. Advanced Pronoun & Tense Quick Reference
While you know the basic pronouns, here is how they pair with the future tense auxiliary verbs for the root *Māḍu* (to do), *Hogu* (to go), and *Baru* (to come).

*   **Naanu** (I) -> *Maadthini* (I will do), *Hogthini* (I will go), *Barthini* (I will come - useful for delivery updates).
*   **Naavu** (We) -> *Maadthivi* (We will do).
*   **Neenu** (You - Casual) -> *Maadtiya?* (Will you do?).
*   **Neevu** (You - Respectful) -> *Maadtira?* (Will you do?).
*   **Avanu** (He) -> *Maadthane* (He will do).
*   **Avalu** (She) -> *Maadthale* (She will do).
*   **Avru** (They / Respectful He/She) -> *Maadthare* (They will do).
*   **Adu** (It - Neuter) -> *Maaduthe* (It will do).
