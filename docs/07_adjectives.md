# 07. Sprint 6: Adjectives & Descriptors

In Kannada, descriptive adjectives generally precede the noun they modify, maintaining the exact same syntactic order as English and Hindi (e.g., *Dodda mane* = Big house = *Bada ghar*). 

This lesson covers high-frequency opposites and essential descriptors for size, quality, and taste.

## 1. Opposites and General Descriptors
These adjectives are crucial for evaluating objects, making purchases, or describing the environment.

| Kannada | English Meaning | Context / Notes |
| :--- | :--- | :--- |
| **Chennagide / Olleyadu** | Good / Nice / Well | A universally useful adjective. Can describe food, weather, health, or objects. |
| **Ketta** | Bad | Used to describe poor quality or negative situations. |
| **Dodda** | Big | Always precedes the noun (e.g., *Dodda car*). |
| **Chikka / Sanna** | Small | Use *Chikka* for objects, *Sanna* can also mean thin/small. |
| **Bisi** | Hot | Used for climate, objects, and food (e.g., *Bisi anna* - Hot rice). |
| **Thannige / Sheeta** | Cold | Used for liquids or weather. (*Sheeta* = Sanskrit *Sheet*) |
| **Hosa** | New | Used for new objects or places (e.g., *Hosa books*). |
| **Hale** | Old | Used for old things. Note the proto-Dravidian 'p' to 'h' shift (from *Pazhaya*). |
| **Bega** | Fast | Very useful to urge auto/cab drivers to speed up (*Bega hogi*). |
| **Nidhana / Mellage** | Slow / Softly | Extremely important when asking someone to drive carefully or speak slower. |
| **Jaasthi / Thumba** | More / Too much | Essential for bargaining (*Thumba jaasthi aithu* - It has become too much). |
| **Kami** | Less | Used to negotiate prices (*Swalpa kami maadi* - Make it a little less). |
| **Sari** | Correct / Okay | Used constantly as an agreement particle to confirm instructions or a price. |
| **Thappu** | Wrong | Used to point out an error. |
| **Swachha** | Clean | Sanskrit root (*Swachh*) |
| **Galiju** | Dirty | Used to complain about hygiene. |
| **Hagura** | Light (Weight) | |

## 2. Sensory Experiences (Taste)
The five primary tastes. These are simple adjectives that can be used directly with nouns or with the verb *ide* (it is).

| Kannada | English Meaning | Example / Context |
| :--- | :--- | :--- |
| **Sihi** | Sweet | "Coffee thumba **sihi** ide." (Coffee is very sweet). |
| **Khara** | Spicy / Hot | "Oota swalpa **khara** beku." (I want the food a little spicy). |
| **Huli** | Sour | "Mosaru **huli** aagide." (The curd has become sour). |
| **Kahi** | Bitter | Used for bitter gourd or medicines. |
| **Uppu** | Salt / Salty | (Also the noun for salt). |
| **Ruchi** | Tasty / Delicious | "Oota thumba **ruchi** aagithu." (Food was very tasty). |

*Note: Complex internal states and physical ailments (like hunger, pain, anger) are typically expressed using the "Experiencer Dative Paradigm" (e.g., 'Nanage novu ide' - To me, there is pain).*
