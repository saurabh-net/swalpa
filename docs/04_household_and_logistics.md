# 10. Household, Groceries, and Logistics

A core lexicon is required to manage the home, instruct domestic help, rent accommodation, and navigate grocery shopping effectively.

## 1. Renting and Administration

| Kannada | English Meaning | Spoken Hindi Parallel | Context / Usage |
| :--- | :--- | :--- | :--- |
| **Baadige** | Rent | | *"Baadige eshtu?"* (How much rent?) |
| **Mungada** | Advance / Deposit | | *"Mungada hattu tingalu."* (10 months advance). |
| **Kararu** | Agreement | Karar | *"Kararu maadi."* (Make the agreement). |
| **Khali** | Empty / Vacate | Khali | *"Mane khali ideya?"* (Is the house available/empty?) |
| **Current** | Electricity | | *"Current hogide."* (Power is gone). |
| **Khate** | Account | Khaata | *"Khate open maadbeku."* (Want to open account). |
| **Sahi** | Signature | | *"Illi sahi maadi."* (Sign here). |
| **Duddu / Hana**| Money | | *"Duddu kodi."* (Give money). |

## 2. Household Chores and Domestic Management
Directing domestic help requires specific vocabulary, particularly distinguishing between different types of washing and cleaning.

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Kasam gudisu** | Sweep the floor | *Kasam* (trash) + *gudisu* (sweep). |
| **Nela oresu** | Wipe the floor | *Nela* (floor) + *oresu* (wipe). |
| **Paathre tolee**| Wash vessels | *Paathre* (vessels) + *tolee* (wash hard objects). |
| **Batte ogey** | Wash clothes | *Batte* (clothes) + *ogey* (wash fabrics). |
| **Kasa haaki** | Throw trash | *Kasa* (trash) + *haaki* (put/throw). |
| **Bagilu tegey** | Open door | *Bagilu* (door) + *tegey* (open). |
