# 09. Sprint 8: Travel, Directions, and Conflict

Navigating Bengaluru traffic requires clear, authoritative directions. The core spatial markers you learned previously (*illi*, *alli*) are combined with these directional commands to form the "Auto-Rickshaw Protocol". Handling interactions on the street, including traffic police or minor conflicts, is also crucial.

## 1. The Auto-Rickshaw Protocol
Before boarding, always establish the destination and the pricing model.

* **Destination Check:** Always ask before boarding: *"[Location] hoguttha?"* (Will it go to [Location]?).
* **Pricing Demand:** Establish the rate: *"Meter haaki"* (Put the meter down) or *"Meter rate kodthini"* (I will pay the meter rate).
* **Refusing Surge Pricing:** *"One-and-half beda"* (I don't want to pay the 1.5x meter charge).

## 2. Navigating the Streets (Directions)
Use these terms with the respectful imperative verbs like *Hogi* (Go) or *Thagonli* (Take) or *Nillisi* (Stop).

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Nēra / Sita** | Straight | *"Nēra hogi"* (Go straight). |
| **Edagade** | Left Side | Made of *Eda* (Left) and *gade* (Side). *"Edagade thagonli"* (Take a left). |
| **Balagade** | Right Side | Made of *Bala* (Right) and *gade* (Side). *"Balagade thagonli"* (Take a right). |
| **Mundhe** | Forward / Ahead | *"Swalpa mundhe nillisi"* (Stop a little ahead). |
| **Hindhe** | Behind / Back | Indicates the rear or reversing direction. *"Hindhe banni"* (Come back/reverse). |
| **Illi** | Here | *"Illi nillisi"* (Stop here). |
| **Alli** | There | *"Alli nillisi"* (Stop there). |
| **Bega hogi** | Go fast | *"Naanu late aagideeni, bega hogi"* (I'm getting late, go fast). |
| **Nidhana hogi** | Go slow | Useful for rough roads or navigating narrow gullies. |

## 3. Distance and Space

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Hathira** | Near | *"Hathira ide"* (It is near). |
| **Doora** | Far | *"Thumba doora"* (Very far). |
| **Olagade** | Inside | Useful for directing delivery personnel to leave items inside a gate. |
| **Horagade / Aache** | Outside | Used to instruct drivers or guests to wait externally. |
| **Munde daari illa** | Dead End | Literally: Forward road is not there / No road ahead. |

## 5. Conflict De-escalation
The goal here is to soothe tension rather than amplify it.

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Summane iri** | Be quiet / Just be | *"Saar, summane iri."* (Sir, just be quiet). |
| **Jasti mathad-bedi**| Don't talk too much | Used to stop an argument from stretching. |
| **Thondre illa / Parvagilla**| No problem / It's okay | Ideal for resolving minor bumps or mistakes. |
| **Nanna thappu illa**| Not my mistake | Firmly defending yourself. |
| **Mosa** | Cheat / Fraud | *"Idu mosa."* (This is cheating). |

## 6. Medical Emergencies
If you need to quickly communicate distress to an auto driver, pharmacist, or bystander, these phrases are universally understood.

| Kannada | English Meaning | Context / Usage |
| :--- | :--- | :--- |
| **Bega hospitalge hogi** | Go to the hospital fast | Instructing an auto or cab driver. |
| **Tale novu ide** | My head is aching | General distress (*Tale* = Head, *Novu* = Pain). |
| **Jvara bandide** | I got a fever | Asking for basic medicine at a pharmacy. |
| **Raktha bartide** | Bleeding is happening | Accidental emergency (*Raktha* = Blood). |
| **Doctor ellidare?** | Where is the doctor? | Finding help in a clinic. |
